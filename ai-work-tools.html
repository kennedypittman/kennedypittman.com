1. SEO Planning: Existing Website

This tool uses AI to analyze actual webpage content. It is helpful when a website already exists, and I want:

  1. Page-by-page SEO keyword assessments
  2. To Check overall keyword coverage
  3. Instant generation of SEO title tags and meta descriptions for all pages

The AI is also told to craft the SEO tags for in-market audiences and what buyers would likely search for.

How to use

  • If the client is paying for SEO work: I can use this to look for gaps in SEO topics, or to identify content misalignment.
  • If we're doing site work with no SEO support, we can use this to create a quick starting point for every page's SEO tags.
