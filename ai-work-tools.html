1. SEO: Pages Analysis

This tool uses AI to analyze actual webpage content. It is helpful when a website already exists, and I want:

  1. Page-by-page SEO keyword/topic assessments

The AI is also told to craft the SEO tags for in-market audiences and what buyers would likely search for.

How to use

There are other keyword ranking tools where you can say, "here are my keywords, how do we rank for them?" This tool is more helpful for planning. It says, "here are my pages, what do you think they're about?" Part of site planning is deciding which page will cover which SEO topic, and this let's me organize thoughts from that direction.

  • If the client is paying for SEO work: I can use this to look for gaps in SEO topics, or to identify content misalignment.
  • If a website client doesn't want to pay for SEO work, we can use this to create a quick draft of every page's SEO tags.

2. SEO: Backlink Priorities

This tool takes a list of domains and prioritizes them for link building efficiency and impact. I get:

  1. A prioritized list of websites to work on
  2. The type of website (directory, blog, etc.)
  3. A suggested link building strategy
  4. Other sortable domain attributes

The priority ranking is based on a formula that includes:

  1. Topical relevance to my website
  2. Domain authority
  3. Difficulty to get a link
  4. How many competitors also get links from this domain
  5. A custom modifier (minor impact)

That custom modifier can be anything I want. A common example would be to prioritize "easy to get" backlinks if SEO time is limited.

Where Do I Get My Domains List?

I can put any domains into the tool that I want. BUT what I typically do is use SEMRush to generate a report of domains that my top competitors get links from. I filter out the ones with very low domain authority. I leverage their work as my shortcut.
